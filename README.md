# Computer Graphics & OpenGL Portfolio

##  Project Overview
This repository showcases my journey in learning **Computer Graphics** using **C++** and the **OpenGL/GLUT** library. It transitions from basic 2D shapes to advanced 3D transformations and real-time user interaction.

##  Projects & Exercises

### 1. Interactive 3D Rotating Cube (Main Project)
- **Description:** A multi-colored 3D cube that rotates in real-time.
- **Interactivity:** Use **UP/DOWN Arrow Keys** to increase or decrease the rotation speed.
- **Key Concepts:** 3D Transformations, Depth Testing (`GL_DEPTH_TEST`), and Keyboard Event Handling.

### 2. Mathematical Circle Generation
- **Description:** Rendering a smooth circle using Sine and Cosine functions.
- **Key Concepts:** Trigonometric vertex generation and `GL_POINTS`.

### 3. Basic Primitives & Shapes
- **Blue Point:** Understanding coordinate systems and point sizes.
- **Stippled Line:** Implementing dashed line patterns using `glLineStipple`.
- **Orange Square:** Working with filled polygons and `GL_QUADS`.

##  Technologies Used
- **Language:** C++
- **Graphics API:** OpenGL
- **Utility Toolkit:** GLUT (Graphics Library Utility Toolkit)

##  How to Run
1. Open the desired `.cpp` file in an IDE configured with OpenGL (e.g., Visual Studio or Code::Blocks).
2. Compile and run to view the graphical output.
3. For the **3D Cube**, use the arrow keys to control the animation.

##  Learning Outcomes
- Mastering the Graphics Pipeline and rendering loops.
- Implementing 2D and 3D geometric transformations.
- Understanding memory management in low-level graphics programming.
